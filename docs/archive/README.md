# Archive

This folder stores files moved out of the repository root or loose scratch folders during supervised cleanup.

Nothing here was deleted. Files were moved to reduce root-level clutter while preserving recovery.

## Folders

| Path | Contents |
| --- | --- |
| `root-files/` | Historical root-level docs and packages. |
| `notes/` | Loose notes moved from scratch folders. |
| `legacy-dirs/` | Legacy or empty root-level directories moved out of the main workspace surface. |

## Review Before Deleting

Do not hard-delete archived files until:

1. The user confirms they are no longer needed.
2. They have stayed archived for at least 30 days.
3. Any useful lessons have been distilled into `docs/`, `teaching/`, or supervised proposals.
4. Regression and governance checks pass after the cleanup.

The 30-day mark only makes an archived item eligible for hard deletion. It does not authorize automatic deletion.
