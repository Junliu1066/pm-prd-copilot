# Interactive HTML Prototype

## Open

- One-file handoff: double-click `standalone.html`.
- macOS: double-click `index.html` or `open-mac.command`.
- Windows: double-click `index.html` or `open-windows.bat`.

## Handoff

- Recommended: send the zip package and ask the receiver to unzip it.
- If sending only one file, send `standalone.html`.
- If sending source files directly, copy the whole folder.
- Keep `index.html`, `styles.css`, and `app.js` in the same directory.
- No install or local server is required.

## Edit Text

- Double-click normal text to edit it in place.
- Press Enter to save or Escape to cancel.
- Use local browser persistence when the prototype needs to keep edits after refresh.
