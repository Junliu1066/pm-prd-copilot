# 验收标准

## 必须包含

- `页面说明`
- `页面跳转关系`
- `原型图层`
- `当前边界：本阶段不输出 PNG，不输出 HTML`
- 产品总览、MVP 范围、泳道图、状态流转、权限矩阵、风险闭环等图表应放在对应章节。

## 不得包含

- `PRD 可视化层`
- `原型图 / 线框图`
- `AI 模型选型`
- 具体项目 ID。
- 绝对路径。
- 测试目的或治理修复过程说明。

## 人工审核

- 样例只能作为 0-1 普通业务 PRD 结构候选，不代表所有项目。
- 后续正式 golden sample portfolio 至少还需要其他类型 PRD 样例。
- 如果样例被项目细节污染，应先回到 proposal，不直接改 stable prompt 或 template。
