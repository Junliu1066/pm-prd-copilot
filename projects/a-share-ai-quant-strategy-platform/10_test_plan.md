# A股 AI 量化策略研究平台测试计划

- 文档状态：Draft / 工程验收版
- 适用范围：P0 MVP + P1 竞技场实验
- 最后更新时间：2026-05-08

---

## 1. 测试目标

确保微信小程序、Web C 端、管理后台、Spring Boot API、积分制度、量化研究链路、AI 风控和策略竞技场满足工程上线前的最低质量要求。

必须守住：

- 不出现收益承诺、荐股、跟单、持仓、买卖点、实时信号、自动下单。
- 积分不可提现、不可返利、不可被描述为收益。
- 量化任务只允许 `research_only` / `simulation_only`。

---

## 2. P0 主路径

| 用例 | 步骤 | 预期 |
|---|---|---|
| 游客访问首页 | 打开首页 | 可进入市场温度、策略列表、风险提示 |
| 游客看市场温度 | 打开市场温度页 | 展示分数、解释、更新时间、数据来源 |
| 游客看策略 | 打开策略列表和详情 | 只能看免费或试看内容 |
| 游客看报告 | 打开报告详情 | 看到试看内容和积分解锁提示 |
| 用户登录 | 微信 mock 或短信登录 | 返回 token 和用户信息 |
| 用户看账户 | 登录后打开我的账户 | 展示昵称、积分余额、风险确认状态 |
| 用户关注策略 | 点击关注 | 成功保存，重复关注不报错 |

## 3. 积分测试

| 用例 | 步骤 | 预期 |
|---|---|---|
| 创建账户 | 新用户登录 | 自动创建 `point_accounts` |
| 测试充值 | 创建 `amountCny=30.00` 订单 | 入账 30 积分，写充值订单和流水 |
| 每日赠送 | 当日第一次签到 | 增加 1 积分 |
| 重复签到 | 当日重复提交 | 不重复赠送，返回已签到 |
| 消耗积分成功 | 解锁 10 积分报告 | 扣减余额、写流水、写权益 |
| 积分不足 | 余额 3 解锁 10 积分报告 | 返回 `POINTS_NOT_ENOUGH` |
| 幂等成功 | 同 key 重复提交同请求 | 返回第一次结果 |
| 幂等冲突 | 同 key 不同请求体 | 返回 `IDEMPOTENCY_REPLAY` |
| 并发扣减 | 同一账户并发消耗 | 余额不为负，只允许符合余额的请求成功 |

## 4. 权限测试

| 用例 | 用户 | 预期 |
|---|---|---|
| 游客调用积分账户 | 未登录 | `UNAUTHORIZED` |
| 游客投票 | 未登录 | `UNAUTHORIZED` |
| 普通用户访问后台 | 普通 token | `FORBIDDEN` |
| 后台无 token | 无 | `UNAUTHORIZED` |
| 未解锁报告 | 登录未消耗积分 | 返回试看或 `CONTENT_LOCKED` |
| 已解锁报告 | 登录且有权益 | 返回完整允许内容 |
| 未确认风险 | 登录但未确认风险 | 关键场景返回 `RISK_CONFIRM_REQUIRED` |

## 5. 量化与市场温度测试

| 用例 | 步骤 | 预期 |
|---|---|---|
| 获取市场温度 | 请求最新快照 | 返回 0-100 分数和分项指标 |
| 无快照 | 清空快照 | 返回空态或明确暂无数据 |
| 创建回测任务 | `runMode=research_only` | 创建成功 |
| 创建仿真任务 | `runMode=simulation_only` | 创建成功 |
| 拒绝实盘模式 | `runMode=live_trade` | 返回 `BAD_REQUEST` |
| 写入净值曲线 | 回测任务完成 | `backtest_equity_points` 有数据 |
| 风控通过 | 回撤正常 | `risk_status=pass` |
| 风控警告 | 回撤过大 | `risk_status=warning` 或 `blocked` |

| 高风险命中 | 命中风险词 | 写 `audit_tasks` |

## 10. 合规文本检查

自动或人工检查以下禁用表达：

- 保证收益。
- 稳赚。
- 推荐买入。
- 立即卖出。
- 跟单。
- 带单。
- 仓位建议。
- 实盘自动交易。
- 券商账户绑定。
- 持仓明细公开。

允许表达：

- 仅供研究。
- 回测不代表未来收益。
- 仿真表现。
- 风险提示。
- 最大回撤。
- 适用行情和失效场景。

## 11. 上线前最低门禁

- `make check-all` 通过。
- 分线程交付时 `THREAD=<branch> make check-boundary` 通过。
- API 文档与实现路径一致。
- 数据库 migration 可在空库执行。
- `./mvnw test` 通过。
- 积分并发测试通过。
- AI 风险词测试通过。
- C 端无已废弃收费话术。
- 关键页面有空态、错误态、权限态。
- 后台敏感操作有日志。
- 内测数据和真实数据隔离。
