# UI Style Direction - demo-project

- Generated at: `2026-04-27T13:50:22+00:00`
- Status: draft for human review
- Selected style: `swiss_utility` / Swiss Utility
- Category: professional
- Description: Restrained grid-first interface style for products that must be scanned, compared, and used repeatedly.

## Tokens
- Palette: #F7F8FA, #FFFFFF, #111827, #4B5563, #2563EB, #16A34A, #E5E7EB
- Typography: Inter or system sans; tabular numbers for metrics
- Radius: 6px
- Density: compact but breathable
- Texture: none
- Layout: 12-column grid, tight section hierarchy, strong table and form rhythm
- Motion: subtle state transitions only

## Component Bias
- tables
- filters
- segmented controls
- forms
- side navigation
- status chips
- charts

## Best For
- SaaS operations
- CRM
- approval workflow
- finance dashboard
- B2B admin
- internal tools

## Avoid For
- brand landing pages needing strong emotional recall
- consumer entertainment
- art/editorial showcases

## Quality Gates
- [ ] Can a repeated user complete the top task without reading decorative copy?
- [ ] Do tables, filters, and actions stay aligned at desktop and laptop widths?
- [ ] Are accent colors reserved for state and action instead of decoration?

_None._
