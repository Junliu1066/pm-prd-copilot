# 二期 Codex 执行文档：个人训练效率增强

## 1. 阶段目标

二期目标是提升个人复练效率和个性化程度。用户可见效果是：学生可按 PPT 页或论文章节训练，查看多轮训练趋势，并针对指定薄弱模块进行专项复练。

## 2. 本期输入

- 一期训练数据和用户反馈。
- PRD 产品包和 `feature_matrix.md` 同步拆分文件中的 V1 功能。
- 全量原型图中的 P7-P11 页面。
- 一期报告中的薄弱模块和复练记录。
- 题库使用情况和重复率数据。

## 3. 本期范围

包含：

- PPT 大纲 / 章节文本输入。
- 按页或按章节生成问题。
- 训练历史趋势。
- 专项复练。
- 题库标签扩展。
- 难度和追问强度配置。
- 导出复习清单。

不包含：

- 导师端、班级管理、机构后台。
- 视频训练。
- 学校规范库。

## 4. 总体框架

| 层 | 二期内容 |
| --- | --- |
| 前端 | PPT/章节输入、趋势页、专项复练入口 |
| 后端 | 历史趋势、章节题目绑定、专项复练服务 |
| 数据 | 增加 PageOutline、TrainingTrend、ModulePractice |
| AI / Prompt / RAG / Memory | 按页提问、趋势解释、专项复练建议 |
| Skill / MCP | 可评估是否新增题库结构化 Skill |
| Harness | 增加重复率和训练历史一致性检查 |
| GitHub / PR | 独立二期 PR |
| 教学 / 记忆 | 复练策略反馈进入项目偏好或通用 lesson |

## 5. 页面与交互

| 页面 | 用户动作 | 系统反馈 | 异常状态 |
| --- | --- | --- | --- |
| PPT/章节输入 | 粘贴 PPT 大纲或章节 | 生成页级问题 | 内容过少提示补充 |
| 历史趋势 | 查看多轮训练变化 | 展示五维变化和模块变化 | 数据不足提示至少完成两轮 |
| 专项复练 | 选择薄弱模块 | 生成专项题单 | 模块题不足时用相近模块 |
| 导出清单 | 导出低分题和复练题 | 输出复习摘要 | 不导出完整论文资料 |

## 6. 服务与数据

| 服务 | 数据对象 | 说明 |
| --- | --- | --- |
| 大纲服务 | PageOutline | 保存 PPT/章节结构 |
| 趋势服务 | TrainingTrend | 汇总多轮评分 |
| 专项复练服务 | ModulePractice | 按模块抽题和记录表现 |
| 导出服务 | ReviewSummary | 导出低风险复习摘要 |

## 7. AI / Prompt / RAG / Memory

- AI 增加“按页提问”和“趋势解释”。
- Prompt 增加章节/PPT 上下文。
- RAG 仍以项目内题库和用户资料为主，不接外部资料。
- Memory 增加多轮训练历史，但必须可删除。

## 8. Skill / MCP 接入

| 类型 | 二期策略 |
| --- | --- |
| Skill | 评估是否新增 `defense-question-bank-normalizer` |
| MCP | 默认不接；如需要读取云文档必须单独批准 |
| Harness | 增加题目重复率和历史趋势数据完整性检查 |

## 9. Codex 任务包

| task_id | goal | allowed_write_paths | forbidden_write_paths | validation | human_confirmation_points |

## 10. 人工确认点

- 是否允许保存多轮训练历史。
- 是否允许导出复习摘要。
- 是否新增题库结构化 Skill。
- 是否接入外部文档输入。

## 11. GitHub / 发布流程

- 分支：`codex/graduation-defense-phase-2`
- 二期 PR 不修改一期已冻结 PRD 范围。
- 发布前需要一期主路径回归。

## 12. Harness / 审计 / 回归

- 一期主路径回归。
- 题目重复率检查。
- 历史趋势一致性检查。
- 导出隐私检查。

## 13. 教学与记忆沉淀

- 用户对复练策略的偏好先进入项目候选偏好。
- 题库标签和按页训练方法若通用，进入 Skill proposal。
- 导出格式偏好如果跨项目适用，进入全局记忆提案。

## 14. 风险与回滚

| 风险 | 回滚 |
| --- | --- |
| 历史趋势误导用户 | 隐藏趋势解释，仅展示原始分 |
| 大纲提问不贴合 | 回退到模块题库 |
| 导出泄露资料 | 暂停导出，仅展示页面内摘要 |

## 15. 验收标准

- 用户可按 PPT/章节训练。
- 至少两轮训练后可展示趋势。
- 可指定薄弱模块复练。
- 导出摘要不包含完整论文资料。
- 一期主路径无回归。

## 16. 下一期衔接

二期输出训练历史、模块表现和复练数据，为三期导师授权查看、班级训练和批量报告提供基础。
