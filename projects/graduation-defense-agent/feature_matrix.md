## 2. MVP 功能矩阵

| MVP 模块 | 必做功能 | 不做功能 | 验收重点 |
| --- | --- | --- | --- |
| 新建训练 | 论文基本资料、资料完整度、训练模式 | 完整文件上传、自动解析 | 2 分钟内创建训练 |
| 问题生成 | 按模式抽题、题目去重、考察点展示 | 专业知识图谱 | 生成不少于 5 题 |
| 模拟答辩 | 问题卡、文字作答、提交回答 | 语音、视频、情绪识别 | 用户能连续完成训练 |
| 动态追问 | 空泛、缺依据、回避不足、答非所问追问 | 无限追问 | 追问相关，最多 2 轮 |
| 五维评分 | 分数、原因、建议 | 真实成绩预测 | 每个低分维度有建议 |
| 示范改写 | 结构化改写、待补充占位 | 编造事实 | 不新增论文事实 |
| 训练报告 | 总览、薄弱模块、典型问题、复练题单 | 高级数据看板 | 报告可直接指导复练 |
| 复练 | 低分模块加权抽题 | 长期能力画像 | 从报告进入复练 |
| 隐私与诚信 | 删除资料、删除记录、高风险拦截 | 学校合规系统集成 | 不代写、不编造、不承诺通过 |

## 3. 页面功能矩阵

| 页面 | 展示内容 | 用户操作 | 系统反馈 | 异常处理 |
| --- | --- | --- | --- | --- |
| P1 新建训练 | 论文资料表单、资料完整度、训练模式 | 填写资料、选择模式、开始训练 | 生成题组或提示资料不足 | 高风险输入触发诚信拦截 |
| P2 模拟答辩 | 老师问题、考察点、回答框、追问区 | 输入回答、提交、跳过追问 | 评分、追问或进入复盘 | 空回答提示重答 |
| P3 单题复盘 | 原回答、扣分原因、示范改写、待补充依据 | 查看改写、补资料、下一题 | 保存单题结果 | 缺事实时显示待补充占位 |
| P4 训练报告 | 五维均分、薄弱模块、典型问题、复练计划 | 开始复练、导出摘要、删除记录 | 创建复练 session | 删除前二次确认 |
| P5 复练训练 | 上轮低分模块、复练题、复练回答框 | 查看上次问题、提交复练 | 更新模块表现 | 题目不足时使用相近模块 |
| P6 资料管理 / 删除 | 已保存资料、删除影响、隐私说明 | 补充资料、删除资料 | 更新完整度或删除资料 | 删除前二次确认 |
| P7 PPT / 章节输入 | PPT 大纲、章节文本、页级问题预览 | 粘贴大纲、选择按页/章节训练 | 生成页级问题 | 内容过少提示补充 |
| P8 历史趋势 | 多轮五维趋势、趋势解释 | 查看趋势、进入专项复练 | 推荐薄弱模块 | 数据不足提示至少两轮 |
| P9 专项复练 | 模块选择、题单设置、相近模块说明 | 选择模块、开始专项复练 | 生成专项题单 | 模块题不足时补相近模块 |
| P10 导出复习清单 | 导出内容、隐私提醒、预览摘要 | 预览、导出 | 生成低风险摘要 | 不导出完整论文资料 |
| P11 题库标签管理 | 标签分布、质检规则 | 保存标签规则 | 更新抽题池 | 缺标签题不进入正式抽题 |
| P12 学生授权 | 授权范围、有效期、撤销说明 | 确认授权、取消 | 生成导师查看权限 | 未授权不可查看 |
| P13 导师任务台 | 班级、训练任务、授权统计 | 发布训练任务 | 学生收到任务 | 未授权学生不共享报告 |
| P14 授权学生报告 | 授权范围内训练摘要、辅导建议 | 查看范围、写备注 | 保存导师备注 | 论文全文默认隐藏 |
| P15 班级批量报告 | 模块分布、完成率、样本提示 | 导出脱敏报告 | 生成班级摘要 | 样本少于阈值不展示细分 |
| P16 语音训练 | 语音问题、录音状态、转文字反馈 | 录音、提交语音、改为文字 | 转写并评分 | 权限拒绝回到文字训练 |
| P17 视频模拟 | 摄像头预览、视频复盘内容 | 开始视频模拟 | 生成视频复盘 | 权限拒绝回到文字训练 |
| P18 学校规范库 / 组织看板 | 学校规范来源、组织质量趋势 | 选择规范、查看脱敏趋势 | 规范化训练或组织分析 | 来源缺失不可使用 |

## 4. 能力依赖矩阵

| 功能 | 依赖题库 | 依赖论文资料 | 依赖 AI | 可规则兜底 | 说明 |
| --- | --- | --- | --- | --- | --- |
| 通用问题训练 | 是 | 否 | 否 | 是 | 资料不足时使用 |
| 个性化问题训练 | 是 | 是 | 是 | 部分 | 不足事实必须提示 |
| 动态追问 | 是 | 是 | 是 | 部分 | 可按固定触发规则兜底 |
| 五维评分 | 是 | 否 | 是 | 部分 | 规则只能给粗粒度提示 |
| 示范改写 | 是 | 是 | 是 | 部分 | 必须防止新增事实 |
| 复练计划 | 是 | 否 | 部分 | 是 | 可按低分模块生成 |
| 学术诚信拦截 | 否 | 否 | 部分 | 是 | 规则优先，模型辅助 |
| 删除资料 | 否 | 否 | 否 | 是 | 确定性能力 |

## 5. 评审关注点

- MVP 是否只做文字训练闭环，不提前扩语音和视频；
- 资料完整度判断是否足够清楚；
- 追问是否“像老师”，但不过度压迫；
- 五维评分是否能帮助用户知道怎么改；
- 示范改写是否严格避免编造事实；
- 训练报告是否能自然引导复练；
- 隐私删除和诚信拦截是否作为 P0。
