## 1. 全局框架

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│ 价小前投研  高频跟踪▼  行情复盘▼  AI助手▼    联系我们▼   搜索框   我的主页 │
├───────────────┬──────────────────────────────────────────────┬───────────────┤
│ 左侧内容留白/ │ 主工作区                                      │ 右侧快捷栏     │
│ 页面边距       │                                              │ 关注股票       │
│               │                                              │ 关注事件       │
│               │                                              │ 热门概念       │
│               │                                              │ 客服/反馈      │
└───────────────┴──────────────────────────────────────────────┴───────────────┘
```

全局规则：

- 顶部导航固定，主工作区根据模块变化。
- 右侧快捷栏固定，承载关注与快捷入口。
- 所有 AI 内容附近展示来源、置信度、更新时间、风险提示。
- 所有行情/财务数据展示来源和更新时间。

## 8. 原型交互说明

- 顶部导航切换一级模块。
- 事件卡片点击进入事件详情。
- 事件详情中的股票和概念标签可跳转。
- 概念中心支持搜索、筛选、排序。
- 个股详情六个 Tab 保持同一个股票上下文。
- 关注动作统一写入右侧快捷栏。
- AI 输出反馈进入审核与质量优化队列。
