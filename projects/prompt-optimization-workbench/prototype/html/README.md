# Prompt 优化工作台 HTML 原型

## 推荐打开方式

- 解压 `prompt-optimization-workbench-html-prototype.zip`。
- 双击 `html/index.html` 打开原型。
- Windows 和 macOS 都可以直接打开，不需要安装依赖，也不需要启动服务。

## 单文件方式

- 如果只方便发送一个文件，请发送 `standalone.html`。
- `standalone.html` 已内联 CSS 和 JS，双击即可打开。

## 源码方式

- `index.html`、`styles.css`、`app.js` 必须放在同一个目录。
- 如果要继续修改原型，请保留整个 `html` 文件夹。

## 编辑方式

- 双击页面中的普通文本即可编辑。
- 按 Enter 保存，按 Esc 取消。
- 点击空白区域也会自动保存。
- 修改内容只保存在当前浏览器本地，不会自动改写源文件。
