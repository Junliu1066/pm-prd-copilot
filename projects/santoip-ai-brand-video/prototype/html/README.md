# Santoip AI 品牌视频生成器 HTML 原型

## 打开方式

解压 zip 后，直接双击 `index.html` 即可打开。Mac 和 Windows 都可以使用。

也可以打开：

- `open-mac.command`
- `open-windows.bat`
- `standalone.html`：单文件备用版

## 可编辑能力

页面中的普通文本支持双击编辑：

- Enter：保存
- 失焦：保存
- Esc：取消

修改内容保存在当前浏览器的 localStorage 中，适合产品经理快速改文案和字段。

## 文件说明

- `index.html`：页面结构
- `styles.css`：视觉样式
- `app.js`：点击跳转、弹窗、编辑、进度等交互
- `prototype_manifest.json`：页面和交互清单
- `prototype_notes.md`：原型说明
- `standalone.html`：已内联 CSS/JS 的单文件版本
